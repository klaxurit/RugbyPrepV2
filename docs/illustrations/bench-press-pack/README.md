# Bench Press Illustration Pack (Prototype)

Prototype de pack d'illustrations pedagogiques pour "Developpe couche".

Fichiers:
- `bench-press-profile.svg` : vue profil (une pose) avec trajectoire et consignes
- `bench-press-dual-view.svg` : vue profil + inset face (coherence de la barre)
- `bench-press-steps.svg` : fiche 3 etapes (haut / descente / poussee)

Objectif:
- privilegier la lisibilite et la coherence biomecanique
- eviter les ambiguities de perspective de la barre en 2D

Notes:
- Les SVG sont modifiables a la main (couleurs, labels, positions)
- On peut ensuite les integrer dans Remotion, dans l'app, ou les exporter en PNG
